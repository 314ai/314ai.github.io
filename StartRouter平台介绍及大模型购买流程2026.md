## 四、大模型购买与使用流程

### 步骤二：充值积分

**建议充值策略**：

### 步骤三：创建API密钥

1. 登录控制台，进入"API密钥"管理页面
2. 点击"创建新密钥"
3. 设置密钥名称（如：生产环境、测试环境）
4. **安全提示**：密钥仅在创建时显示一次，请立即保存至安全位置

### 步骤四：调用模型API

**标准调用格式**：

```python
import requests

API_KEY = "your-api-key"
BASE_URL = "https://qingfengshuyu.top/api/v1"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

# 对话模型示例
response = requests.post(
    f"{BASE_URL}/chat/completions",
    headers=headers,
    json={
        "model": "claude-3.5-sonnet",  # 或 gpt-4o, gemini-pro 等
        "messages": [{"role": "user", "content": "你好，请介绍一下自己"}],
        "temperature": 0.7
    }
)

print(response.json()["choices"][0]["message"]["content"])
```

**图像生成示例**：

```python
response = requests.post(
    f"{BASE_URL}/images/generations",
```
