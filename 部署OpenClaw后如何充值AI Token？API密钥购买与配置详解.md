## 三、充值购买Token（PTC积分）

302AI采用**PTC积分制**（Platform Token Credit），1美元=1 PTC，支持支付宝/微信等国内支付方式。

### 充值流程：

1. **进入充值页面**  
  登录后台 → 点击左侧菜单"充值"或右上角"余额" → 选择"购买积分"
  
2. **选择充值金额**
  
  - 最低充值：5美元（5 PTC）
  - 建议首次充值：5-10美元即可满足1-2个月个人使用
3. **完成支付**  
  选择支付宝/微信支付，扫码完成付款。充值即时到账，余额显示在控制台首页。
  

> **计费说明**：平台按实际消耗的Token数量扣费，不同模型价格不同。例如：
> 
> - GPT-4o：输入$2.5/1M tokens，输出$10/1M tokens
> - Claude 3 Opus：输入$15/1M tokens，输出$75/1M tokens
> - DALL-E 3：$0.04/次（1024x1024）

## 四、生成API Key

充值完成后，需要创建API密钥供OpenClaw调用。

### 操作步骤：

1. **进入API管理**  
  控制台左侧菜单 → **"API Keys"** → 点击"添加API"
  
2. **创建密钥**
  
  - 名称：自定义（如"OpenClaw-Home"）
  - 权限：建议保持默认（全部权限）
  - 点击"添加"生成密钥
3. **复制密钥**  
  生成后点击"查看"按钮，复制完整的API Key（格式类似`sk-302-xxxxxxxx`）。  
  ⚠️ **重要**：密钥仅显示一次，请务必保存到安全位置！

## 五、OpenClaw配置302AI的三种方式

根据你的使用场景，选择以下任一方式配置：

### 方式一：命令行快速配置（推荐）

302AI官方提供了`302oc`配置工具，1分钟完成OpenClaw配置：

```bash
# 使用npx直接运行（无需安装）
npx 302oc
```

**配置流程**：

1. 输入刚才复制的302AI API Key
2. 选择API节点（国内用户选"国内节点"）
3. 选择默认AI模型（推荐Claude 3.5 Sonnet或GPT-4o）
4. 选择界面语言（中文）
5. 选择"应用配置并退出"

配置完成后，OpenClaw会自动使用302AI的API服务。

### 方式二：OpenClaw初始化向导配置

如果你是首次运行OpenClaw，使用内置向导配置：

```bash
openclaw onboard --install-daemon
```

当向导询问API Key时：

- **AI模型**：选择"Anthropic"（即使使用GPT也选此项，302AI兼容接口）
- **API Key**：粘贴你的302AI API Key
- **Base URL**（如有选项）：填写 `https://api.302.ai/v1`

### 方式三：手动配置（已部署用户）

如果OpenClaw已运行，需手动修改配置文件：

1. **找到配置文件**  
  配置文件通常位于：
  
  - Windows: `%APPDATA%\OpenClaw\settings.json`
  - macOS: `~/Library/Application Support/OpenClaw/settings.json`
  - Linux: `~/.config/OpenClaw/settings.json`
2. **修改API参数**  
  编辑文件，修改以下字段：
  
  ```json
  {
    "llm": {
      "provider": "openai",
      "api_key": "sk-302-你的API密钥",
      "base_url": "https://api.302.ai/v1",
      "model": "gpt-4o"
    }
  }
  ```
  
3. **重启OpenClaw服务**
  
  ```bash
  openclaw gateway --port 18789 --verbose
  ```

## 九、常见问题与解决方案

| 问题  | 原因  | 解决方案 |
| --- | --- | --- |
| 提示"API Key无效" | 密钥复制错误或过期 | 重新生成API Key，检查是否包含空格 |
| 接口连接超时 | 网络问题 | 切换302AI的国内/国际节点，或配置代理 |
| 余额充足但无法调用 | 模型权限未开通 | 检查该模型是否需要额外申请 |
| OpenClaw配置后未生效 | 配置文件未正确保存 | 使用`302oc`工具重新配置，或重启OpenClaw服务 |
| 模型返回内容受限 | 内容安全策略 | 302AI部分模型有内容审核，尝试切换其他模型 |

**接口报错专属解决**（针对Nano Banana等特定模型）：
如果提示接口连通失败，手动更新接口地址：

```
技能 302.AI 的请求接口，请更新成这个：https://api.302ai.cn/ws/api/v3/google/nano-banana/text-to-image
```
