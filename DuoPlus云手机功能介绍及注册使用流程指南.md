## 二、核心功能详解

- **100+设备机型**：提供Google、三星、OPPO、vivo等主流品牌机型选择

### 2. 多账号批量管理

针对跨境电商和社媒矩阵运营需求，DuoPlus提供强大的批量管理功能：

- **无限多开**：无购买上限、无多开上限、无打开次数限制

### 3. 团队协作与分享

- **团队共享**：支持一键加入团队，多人高效管理操作云手机
- **链接分享**：可将云手机分享给任意用户，对方无需DuoPlus账号，通过链接和密码即可操作
- **权限管理**：支持子成员分享权限控制，优化团队内云手机权限分配

### 4. 直播与自动化功能

- **无人直播**：支持OBS推流和视频文件播放，实现24小时无人值守直播

- **API接口**：提供REST API接口，支持ADB命令执行，适合大规模规则化操作

DuoPlus提供配套的云号码功能，用户可购买海外实体号或虚拟号，在线接收验证码完成账号注册，实现完整的"设备+号码"解决方案。

## 五、注册与使用流程

### 步骤一：账号注册

1. 访问DuoPlus官网：<https://www.duoplus.cn/share/boss123>
2. 点击"立即注册"，支持两种方式：
  - **邮箱注册**：填写邮箱、设置密码，通过验证码完成注册
  - **谷歌登录**：直接使用Google账号快捷登录
3. 新用户注册后可获得免费试用时长

### 步骤二：购买云手机

1. 登录后台，点击右上角"购买云手机"或列表中的"立即体验"
2. 选择购买数量（1-500台可选）和时长（月付/年付）
3. 选择支付方式：
  - **余额支付**：从账户余额扣除，无手续费（充值金额<$100时收取手续费）
  - **现金支付**：支持微信、支付宝、PayPal（单笔<$100时收取手续费）

### 步骤三：配置代理IP（关键步骤）

**重要提示**：DuoPlus云手机**必须配置SOCKS5代理**才能正常使用，不支持HTTP/HTTPS代理。

1. 购买完成后进入代理配置页面（或点击云手机列表的"修改参数"）
2. 在"IP或域名"输入框粘贴代理信息，系统自动识别格式：
  - IPv4：`主机:端口` 或 `主机:端口:账号:密码` 或 `账号:密码@主机:端口`
  - IPv6：`[主机]:端口` 或 `[主机]:端口:账号:密码` 或 `账号:密码@[主机]:端口`
3. 点击"检查代理"，显示"代理检测通过"后点击"完成配置"
4. **验证代理**：开机后在浏览器访问ipinfo.io，确认IP地址与配置一致

### 步骤四：开机使用

1. 在云手机列表点击"开机"按钮
2. 等待30秒左右，云手机启动成功
3. 点击"进入设备"，即可在浏览器中操作云手机
